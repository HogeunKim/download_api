# Prompt Library

이 폴더는 Cursor에서 자주 사용하는 프롬프트를 파일로 저장해 재사용하기 위한 공간입니다.

## 사용 방법

1. `00_template.md`를 복사해서 새 파일을 만듭니다.
2. 작업 목적에 맞게 입력값/출력 형식을 작성합니다.
3. Cursor 채팅 입력창에 파일 내용을 붙여 넣어 사용합니다.

## 권장 규칙

- 파일명은 `NN_주제.md` 형태로 작성
- 한 파일에는 한 가지 작업 목적만 작성
- 프로젝트 규칙(.cursorrules)과 충돌하지 않게 유지

## 예시 파일

- `10_download_api_test.md`
- `20_docker_deploy_ubuntu.md`
